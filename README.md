# zhpy
周蟒, 用中文化 python 語法寫程式

## 安裝

$　pip install -U zhpy

## 使用

安裝好後你可以立即試用周蟒中文 Python 直譯器. 試著在直譯器上輸入

```
印出 '哈囉, 世界'
```

```
$ python interpreter.py
zhpy X.X in darwin on top of Python 2.X.X
>>> 印出 '哈囉, 世界'
哈囉, 世界
```

這樣你就完成了你的第一個中文程式囉!
